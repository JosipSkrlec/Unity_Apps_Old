SpaceShips



INFO:

Project is created in Unity Game engine 5.6

All 3d content is created in Blender 2.79 and textures inside

All code is written in c# programming language


FEATURES:

-sound (downloaded form freesound.org)

-in-game upgrades

-game Save/Load system

-Game connect to server with PHP

-multilanguage system

-Day/night system, Particle system


Game Controls:

move - (W,A,S,D)

light - hold 'G'

car horn - hold 'G'

turbo - 'space'
